# Projekt_Semester_4
Projekt im 4.Semester Studiengang: Systems Engineering

Ziel: Luftqualitätsanalyse in Büro-/Arbeitsbereichen

Dafür nötige Hardware-Komponenten:

  - Raspberry Pi
  - ADS1115 (Analog-Digital Wandler)
